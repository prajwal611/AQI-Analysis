# AQI-Analysis
🌍 AQI Analysis Dashboard (Python + SQL + Power BI)
📖 Overview

This project analyzes Air Quality Index (AQI) data using Python, SQL, and Power BI to monitor pollution levels, identify trends, and generate actionable insights. The goal is to understand environmental patterns and support data-driven decisions for improving air quality.

🎯 Objectives
Analyze AQI trends over time
Identify most polluted cities and regions
Study the impact of pollutants (PM2.5, PM10, NO₂, CO, SO₂)
Perform data cleaning and transformation
Build an interactive dashboard for visualization
🛠️ Tools & Technologies
Python (Pandas, NumPy, Matplotlib, Seaborn)
SQL (Data extraction, querying, aggregation)
Power BI (Dashboard & visualization)
Jupyter Notebook
📂 Dataset

The dataset includes air quality data with attributes such as:

Date
City
AQI Value
PM2.5, PM10
NO₂, SO₂, CO
AQI Category (Good, Moderate, Poor, etc.)
🔍 Project Workflow
🐍 Python (Data Processing & EDA)
Data cleaning (handling missing values, duplicates)
Exploratory Data Analysis (EDA)
Trend analysis and pollutant impact study
Data preparation for SQL and Power BI
🗄️ SQL (Data Querying & Analysis)
Extracted insights using SQL queries
Aggregations (average AQI, city-wise pollution)
Time-based analysis (monthly, yearly trends)
📊 Power BI (Dashboard)
Interactive dashboard with filters (City, Date, Pollutant)
KPIs: Average AQI, Highest AQI, Pollution Levels
Visualizations:
AQI trends over time
City-wise AQI comparison
Pollutant contribution analysis
📈 Key Insights
Identified cities with consistently high pollution levels
Observed seasonal trends in AQI
Found major pollutants contributing to poor air quality
Highlighted critical periods of severe pollution
🚀 How to Use
Run Python scripts / Jupyter Notebook for preprocessing
Execute SQL queries for analysis
Open Power BI .pbix file for dashboard visualization
Use filters to explore different insights
